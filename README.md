# Foto Kolaj

İki fotoğrafı örüntülü kolaja dönüştüren web aracı. Çıktı boyutu: **1080 × 1350 px** (Instagram dikey).

## Kullanım

```bash
cd fotoduzen
python3 -m http.server 8080
```

Tarayıcıda aç: http://localhost:8080

1. İki fotoğraf yükleyin (JPG/PNG)
2. Büyük önizlemede bir bölüme tıklayıp sürükleyerek hizalayın
3. **İndir (PNG)** ile kaydedin

> Dosyayı doğrudan açmayın — mutlaka `python3 -m http.server` ile çalıştırın.
